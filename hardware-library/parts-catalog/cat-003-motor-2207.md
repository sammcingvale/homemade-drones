## Abstract spec

- 2207 stator (22mm diameter, 7mm height)
- 1750–2000KV (matched to 4S battery, 5" prop)
- M5 prop shaft thread (standard for 22xx motors)
- 16×16mm and 19×19mm motor mount hole patterns (Source One V5 arms use 16×16 with M3)
- N52 magnets minimum
- Continuous current rating ≥30A on 4S
- Pre-soldered motor leads (~150mm)

## Notes

- Always order spares with new builds. Crashes destroy motors silently (bent shafts, demagnetized stators).
- **Motor direction is set in firmware**, not by motor SKU. CW/CCW labels on motors refer to which way the bell spins, but in BLHeli configurator we flip direction as needed. Don't worry about CW/CCW pairs at order time.
- Motors run hotter than 70°C at hover = something is wrong (prop binding, ESC mistuned, motor mount not flat). Investigate before flying again.
