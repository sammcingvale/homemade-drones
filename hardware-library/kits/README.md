# Kits

Packing lists for specific operations. A kit is a *checklist of what to bring*, useful when going to the field or starting a build session.

## Available kits

| Kit | Purpose |
|---|---|
| `v1-field-kit.md` | What to bring to a field flight session for the v1 airframe |

## When to add a new kit

When you find yourself packing the same set of stuff twice. Examples that may emerge:

- `v1-soldering-bench.md` — what should be on the bench when working on v1 electronics
- `crash-recovery.md` — what to bring when retrieving a crashed airframe
- `multi-airframe-field.md` — when running 2+ airframes in the same session
